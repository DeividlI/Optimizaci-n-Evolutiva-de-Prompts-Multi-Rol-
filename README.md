# Optimizaci-n-Evolutiva-de-Prompts-Multi-Rol-
Sistema de optimización automática de prompts usando algoritmos genéticos y LLMs locales (Ollama GPT-OSS:20B). Combina hasta 5 roles simultáneos para maximizar la calidad de respuestas en sistemas Q&amp;A universitarios, evaluando similitud semántica mediante embeddings de SentenceTransformer.
